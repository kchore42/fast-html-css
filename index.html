HTML(전체 범위) 는 HTML 문서의 전체 범위를 지정합니다. 웹 브라우저가 해석해야 할 HTML 문서가 어디에서 시작하며, 어디에서 끝나는지 알려주는 역할을 합니다. HEAD(정보 범위) 웹 브라우저가 해석해야 할 HTML 문서의 정보 범위를 지정합니다. 여기서 말하는 정보에는 웹 페이지의 제목, 웹 페이지의 문자 인코딩 방식, 연결할 외부 파일의 위치, 웹 페이지를 구조화하기 위한 기본 세팅 값 같은 것들을 말합니다. 다르게는 ‘화면을 구성하기 위한 기본 설정’이라고 표현할 수 있습니다. BODY(구조 범위) 웹 브라우저가 해석해야 할 HTML 문서의 구조 범위를 지정합니다. 구조는 사용자가 화면을 통해서 볼 수 있는 내용(콘텐츠)의 형태나 레이아웃 등을 의미하며 로고, 헤더, 푸터, 내비게이션, 메뉴, 버튼, 입력창, 팝업, 광고 등 보이는 모든 것들이 구조에 해당합니다. 구조는 BODY 범위 안에서만 생성합니다. DOCTYPE(DTD, 버전 지정) DOCTYPE(DTD, Document Type Definition)은 마크업 언어에서 문서 형식을 정의합니다. 이는 웹 브라우저에 우리가 제공할 HTML 문서를 어떤 HTML 버전의 해석 방식으로 구조화하면 되는지를 알려줍니다. (HTML은 크게 1, 2, 3, 4, X-, 5 버전이 있습니다) 현재의 표준 모드는 HTML5 입니다. Windows 운영체제가 95, 98, ME, XP, Vista, 7, 8, 10 버전이 있는 것과 비슷하다고 생각하시면 쉽습니다.
